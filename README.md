# Crocs Website Automation Testing

This project automates the testing of the Crocs e-commerce website using Python and Selenium WebDriver. It verifies important user workflows such as searching for products, adding items to the cart, login validation, and checkout navigation.

## Project Objectives

- Automate repetitive website testing
- Validate key user functionalities
- Improve testing efficiency
- Detect UI navigation issues

---

## Test Scenarios

- Website Launch
- Home Page Validation
- Product Search
- Product Details Verification
- Add Product to Cart
- Login Validation
- Checkout Navigation
- Screenshot Capture on Test Execution

---

## Technologies Used

- Python
- Selenium WebDriver
- ChromeDriver
- HTML
- CSS

## Installation

Clone the repository

```bash
git clone https://github.com/prajwalhk560-cyber/Crocs-Website-Automation-Testing.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the script

```bash
python tests/test_search.py
```
